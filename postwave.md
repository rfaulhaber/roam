# postwave

tags
: [[my projects]]

A Tumblr clone.


## Desired features

-   OAuth API
-   Federated
-   FOSS
-   Search and tag feature that works well
-   REST and GraphQL API
-   Locked blogs
-   Non-rebloggable posts
-   Password protected posts
-   Performant
-   Filter dashboard, including but not limited to:
    -   By original post
    -   By type
    -   By set of users
-   useful notifications
